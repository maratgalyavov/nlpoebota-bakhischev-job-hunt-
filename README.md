# HR Career Assistant 🎯

## О проекте
**HR Career Assistant** — умный чат-бот, который помогает соискателям найти работу мечты. Он проводит **персональное «интервью»**, на основе ответов формирует **структурированное резюме** и **векторный профиль пользователя**, затем подбирает **релевантные вакансии** из базы (по embedding similarity), помогает составить **сопроводительное письмо** и показывает, **каких навыков не хватает** для желаемой позиции.

Поиск работы — это долго и стрессово. Нужно мониторить десятки сайтов, адаптировать резюме под каждую вакансию, писать сопроводительные письма. Данный чат-бот позволяет автоматизировать рутину, дать персонализированный подбор и честную обратную связь по навыкам.

## Функционал чат-бота
- Проводит пошаговое интервью с пользователем (опыт, навыки, образование, предпочтения)
- Генерирует готовое резюме в структурированном формате
- Подбирает топ-N вакансий по схожести профиля (k-NN в embedding-пространстве)
- Составляет сопроводительное письмо под конкретную вакансию
- Показывает skill gaps — каких навыков не хватает + ресурсы для обучения
- Отмечает % match с вакансией и ключевые требования

## NB
Бот не гарантирует трудоустройство. Это помощник для поиска и подготовки — финальное решение и проверка информации остаётся за пользователем.

## Стек технологий
- **Backend:** Python, FastAPI
- **LLM:** Open-source LLMs (e.g. Qwen3.5-Flash)
- **Embeddings:** sentence-transformers 
- **Vector DB:** FAISS / ChromaDB
- **Bot Interface:** Telegram Bot API
- **Storage:** SQLite (user data), Docker (containerization)
